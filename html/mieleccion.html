Motivos para escoger Jira:

Empresarial

Acostumbrarnos a un software de uso corporativo

Control de Versiones

Conectividad con BitBucket, importante dado que es el Github de las empresas

Sprints

Plantillas cómodas para la generación de sprints, lo cual facilitará el desarollo de el proyecto en común

IDE

Conectividad con VsCode para ver en tiempo real y de forma más cómoda cuando cualquier miembro realiza un cambios
