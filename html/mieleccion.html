Motivos para escoger Jira:

Empresarial

Acostumbrarnos a un software de uso corporativo

Control de Versiones

Conectividad con BitBucket, importante dado que es el Github de las empresas

Sprints

Plantillas cómodas para la generación de sprints, lo cual facilitará el desarollo de el proyecto en común

IDE

Conectividad con VsCode para ver en tiempo real y de forma más cómoda cuando cualquier miembro realiza un cambios

Seguro

Capacidad para otorgar permisos y asegurar así la seguridad de nuestro desarrollo

Apoyo

Gran comunidad con miles de tutoriales para resolver cualquier duda con proyectos de ejemplo en la propia documentación

Eficacia

Capacidad para poder asignar las tareas de forma individual y que quede así más claro que debe hacer cada uno

Diseño

Conexión con Figma y multiples herramientas que facilitan el desarrollo
